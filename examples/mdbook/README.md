# mdBook example

Copy `theme/highlight-sui-move.js` into your book’s `theme/` folder, add it to `book.toml` under `[output.html] additional-js`, and pin the jsDelivr URL to the `highlightjs-sui` version you use.

See the full explanation in **[`docs/mdbook-integration.md`](../docs/mdbook-integration.md)** (including the **Highlight.js 10 vs 11** `highlightBlock` / `highlightElement` note).
